# my-mui-lib

> Created To Test Library for Global Access

[![NPM](https://img.shields.io/npm/v/my-mui-lib.svg)](https://www.npmjs.com/package/my-mui-lib) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save my-mui-lib
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'my-mui-lib'
import 'my-mui-lib/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
